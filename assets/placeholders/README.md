# 📁 Placeholder Images

Drop your brand images into this folder using the exact filenames below.
When you push to GitHub, Shopify's GitHub integration will auto-sync them.

You can also upload/swap images directly in the **Shopify Theme Customizer** — no code needed.

## Required Images

| Filename | Description | Recommended Size |
|---|---|---|
| `hero-desktop.jpg` | Hero background — desktop | 1920×1080 px |

## Notes

- Use JPG for photos, PNG for logos with transparency
- Keep file sizes reasonable: photos under 500KB, logos under 100KB
- Shopify will automatically generate responsive sizes
